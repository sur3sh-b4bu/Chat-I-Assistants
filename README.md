# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1BWRGDAr41Pi6QQ-0MajV_dZXT8HNNZYl

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## Quick Start (Windows)
Double-click the `run_app.bat` file in this directory. 

This script will:
1. Check if Node.js is installed (and warn you if it's missing).
2. Install necessary dependencies (first time only).
3. Start the application and open it in your default browser.
